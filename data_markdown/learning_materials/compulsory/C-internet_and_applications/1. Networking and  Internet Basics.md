# Accessing Resources

Access resources using easily memorable names instead of complex numeric addresses.

## HTTP (Hypertext Transfer Protocol)

**Definition:** HTTP is the protocol used to transfer web pages and other resources on the internet. It operates at the application layer and defines the format and method of exchanging messages between clients (such as browsers) and servers.

## HTTPS (Hypertext Transfer Protocol Secure)

**Definition:** HTTPS is the secure version of HTTP. It uses the TLS/SSL encryption protocol to protect the data transmitted between the user's browser and the server, ensuring the confidentiality and integrity of the data. HTTPS is widely used for websites that require high security, such as online banking and e-commerce platforms.
