# Virtual Machines

## Overview

Virtual Machines (VMs) are software emulations of physical computers that run an operating system and applications just like a physical machine. They allow multiple operating systems to run on a single physical server.

### Components

- **Apps**: Various applications that can run on the virtual machines.
- **OS**: The operating systems that are installed on the virtual machines.

### Architecture

```
Virtualization Server
├── Physical Server 1
├── Physical Server 2
└── Physical Server N
└── Shared Storage
```
