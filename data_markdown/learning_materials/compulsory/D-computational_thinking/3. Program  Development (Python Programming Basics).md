## Example:

```python
numbers = [2, 3, 5, 6, 8, 10]
is_sorted = True

for i in range(len(numbers) - 1):
if numbers[i] > numbers[i + 1]:
is_sorted = False
break

print(f"Is the list sorted? {is_sorted}")
```

### Explanation:

We assume the list is sorted and iterate through it to check if each element is less than or equal to the next. If we find an unordered element, we set `is_sorted` to `False`.
