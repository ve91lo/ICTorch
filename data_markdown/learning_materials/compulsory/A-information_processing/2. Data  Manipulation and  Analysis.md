# 6. DATA TYPES

Data types specify the format and purpose of data. Common data types include:

- **Character**
A single letter, number, or symbol.
Example: `'a'`, `''`.

- **String**
A sequence of multiple characters.
Example: `"Hello World"`.

- **Integer**
A number without a decimal part.
Example: `1`, `2`, `3`.

- **Real Number**
A number that includes a decimal part.
Example: `1.1`, `2.0`.

- **Boolean**
A logical type with only two values.
Example: `TRUE` (true) or `FALSE` (false).

- **Date**
Used to represent dates and times.
Example: `"2023-10-01"`.
