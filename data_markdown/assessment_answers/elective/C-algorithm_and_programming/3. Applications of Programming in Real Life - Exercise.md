# Accelerometers in Fitness Tracking

Accelerometers are commonly used in fitness tracking applications to measure movement and activity levels. They can detect changes in motion and orientation, allowing applications to track steps, estimate distances, and monitor overall physical activity. For example, an accelerometer in a fitness tracker can identify when a user is walking or running based on the intensity and pattern of movement.

## Python Code Example

```python
import time
import random

def detect_activity(x, y, z):
magnitude = (x**2 + y**2 + z**2) ** 0.5
if magnitude > 1.2:
return "Walking or Running"
else:
return "Stationary"

while True:
x = random.uniform(-2.0, 2.0)
y = random.uniform(-2.0, 2.0)
z = random.uniform(-2.0, 2.0)

activity = detect_activity(x, y, z)
print(f"Accelerometer Data - X: {x:.2f}, Y: {y:.2f}, Z: {z:.2f} => Activity: {activity}")

time.sleep(2)  # Check every 2 seconds
```
