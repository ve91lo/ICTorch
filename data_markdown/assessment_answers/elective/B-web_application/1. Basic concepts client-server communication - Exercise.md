# Dynamic Host Configuration Protocol (DHCP)

A DHCP (Dynamic Host Configuration Protocol) server automatically assigns IP addresses and other network configuration parameters to client devices on a network. This simplifies network management by dynamically providing IP addresses, avoiding IP conflicts, and ensuring proper network configuration.

## 1.6. How Do Proxy Servers Enhance Network Performance and Security?

**Answer:**

Proxy servers act as intermediaries between client devices and the internet. They can provide caching, content filtering, and user anonymity. By forwarding client requests to the internet and returning responses, proxy servers improve performance through caching and enhance security by monitoring and controlling internet usage.

## 1.7. What is the Significance of Port Numbers in TCP Communication? Provide Examples

**Answer:**

Port numbers in TCP communication identify specific communication endpoints and services. They ensure that data is directed to the correct application. Common examples include:

- Port 80 for HTTP traffic
- Port 443 for HTTPS traffic

These port numbers help manage and route data correctly between client and server applications.
