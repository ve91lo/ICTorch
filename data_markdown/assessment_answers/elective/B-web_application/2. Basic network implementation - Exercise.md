# ICT Educational Material

## 1.11. When Setting up Security Policies for a Temporary Network, how Can One Balance Convenience and Protection?

**Answer:**

Balancing convenience and protection can be achieved by implementing layered security measures, such as:

- Using strong encryption standards
- Creating guest networks to isolate temporary network traffic
- Implementing temporary access control policies
- Ensuring that network security configurations are easy to manage

Additionally, providing user-friendly access methods (e.g., simplified login processes) can enhance convenience.

## 1.12. When Setting up Database Services, how Can One Effectively Manage Database Security and Performance?

**Answer:**

Effective management of database security and performance includes:

- Setting appropriate user permissions and encryption measures to protect data
- Regularly backing up data to prevent loss
- Implementing performance monitoring to check and adjust query performance
- Optimizing database configurations (e.g., indexes and query plans) to improve efficiency
- Performing regular maintenance and updates to ensure database stability
