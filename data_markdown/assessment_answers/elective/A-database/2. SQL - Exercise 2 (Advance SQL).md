# SQL Queries

## 1.8. Write a query to find the FirstName and LastName of all employees whose last names start with the letter 'J'.

**Answer:**
```sql
SELECT FirstName, LastName
FROM Employees
WHERE LastName LIKE 'J%';
```

## 1.9. Write a query to find the FirstName and LastName of all employees whose last names contain exactly five letters.

**Answer:**
```sql
SELECT FirstName, LastName
FROM Employees
WHERE LastName LIKE '_____';
```

## 1.7. Write a query to find the DeptID and the count of employees in each department where the count is less than 3.

**Answer:**
```sql
SELECT DeptID, COUNT(EmpID) AS EmployeeCount
FROM Employees
GROUP BY DeptID
HAVING COUNT(EmpID) &#x3C; 3
ORDER BY EmployeeCount ASC;
```
