## 1.9. QUERY THE TOTAL NUMBER OF EMPLOYEES ASSIGNED TO EACH PROJECT

**Answer:**

```sql
SELECT p.project_name, COUNT(a.employee_id) AS number_of_employees
FROM projects p
JOIN assignments a ON p.project_id = a.project_id
GROUP BY p.project_name;
```
