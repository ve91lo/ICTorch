# SQL Queries for Employee Database

## 1.10. Write a query to count the total number of employees in the Employees table

**Answer:**
```sql
SELECT COUNT(*) FROM Employees;
```

## 1.11. Write a query to select FirstName, LastName, and HireDate of employees, ordered by HireDate in descending order

**Answer:**
```sql
SELECT FirstName, LastName, HireDate FROM Employees ORDER BY HireDate DESC;
```
