# SQL Queries

## 1.8. Retrieve Employee Names and Their Department Names

**Task:** Write an SQL query to retrieve employee names and their department names, sorted by employee name in ascending order.

**Tables:**

- **Employees** (EmpID, Name, DeptID)
- **Departments** (DeptID, DeptName)

**Answer:**

```sql
SELECT Employees.Name, Departments.DeptName
FROM Employees
INNER JOIN Departments
ON Employees.DeptID = Departments.DeptID
ORDER BY Employees.Name ASC;
```

## 1.9. Find Product Names and Their Categories

**Task:** Write an SQL query to find product names and their categories for products in the 'Electronics' category that include 'Laptop' in the product name.

**Tables:**

- **Products** (ProductID, ProductName, CategoryID)
- **Categories** (CategoryID, CategoryName)

**Answer:**

```sql
SELECT Products.ProductName, Categories.CategoryName
FROM Products
INNER JOIN Categories
ON Products.CategoryID = Categories.CategoryID
WHERE Categories.CategoryName = 'Electronics' AND Products.ProductName LIKE '%Laptop%';
```
