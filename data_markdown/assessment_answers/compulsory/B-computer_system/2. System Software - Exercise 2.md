# ICT Educational Material

## Questions and Answers

### 28. What system would be most appropriate for monitoring and controlling industrial machinery in a manufacturing plant?

**Answer:**

A real-time system would be most appropriate for monitoring and controlling industrial machinery, as it requires immediate processing of sensor data to ensure safety and efficiency in operations.

### 29. In a scenario where large datasets are analyzed for research purposes, which type of system should be used?

**Answer:**

A parallel processing system should be used for analyzing large datasets in research, as it can efficiently handle complex calculations and data processing by utilizing multiple processors simultaneously.

### 30. For a cloud-based application that needs to provide services to users in different geographical locations, what type of system would be ideal?

**Answer:**

A distributed processing system would be ideal for a cloud-based application serving users in different geographical locations, as it can effectively manage resources and data across multiple servers and locations, ensuring reliability and scalability.
