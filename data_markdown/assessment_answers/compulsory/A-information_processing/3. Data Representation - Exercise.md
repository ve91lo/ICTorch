# Choosing a Digital Media Format

When choosing a digital media format, consider the following factors:

## Factors to Consider

1. **File Size**:
- Lossy formats (e.g., JPG, MP3) typically have smaller file sizes compared to lossless formats (e.g., PNG, WAV).

2. **Quality**:
- Lossless formats retain original quality, while lossy formats may lose some quality.

3. **Compatibility**:
- Some formats are more widely supported (e.g., TXT, PDF) than others, which might be specific to certain applications.

4. **Functionality**:
- Formats like DOCX and ODT support complex document features, whereas simpler formats like TXT do not.
