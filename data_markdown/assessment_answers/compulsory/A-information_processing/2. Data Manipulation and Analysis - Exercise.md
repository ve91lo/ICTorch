# ICT Educational Material

## 6.1. Difference Between Integers and Real Numbers

An integer is a number without a decimal part, such as 1, 2, 3, etc.; while a real number can include a decimal part, such as 1.1 or 2.0. Integers are typically used for counting or quantities, whereas real numbers are used for calculations that require precision, such as measurements or monetary amounts.

## 6.2. Representation of Boolean Values in Programming

Boolean values are used to represent true or false, typically denoted as `TRUE` and `FALSE`. For example, in a conditional statement, when a condition is met, it can be expressed as:

```plaintext
isValid = TRUE
```

to indicate that the condition is true; otherwise, it can be expressed as:

```plaintext
isValid = FALSE
```

## 6.3. Use of "String" vs "Character" in Text Data

A "string" is a sequence made up of multiple characters, suitable for storing and processing text data, such as "Hello World." In contrast, a "character" only represents a single character and cannot fully convey a segment of text or information. Therefore, when dealing with text that requires multiple characters, using a string is more appropriate and convenient.
