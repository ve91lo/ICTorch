# QUESTIONS

## 1.1. Draw an ER Diagram to Represent This Scenario

A university database needs to keep track of students, courses, and enrollments.
Each student has a unique student ID, name, and major.
Each course has a unique course ID, course name, and the instructor teaching it.
Each student can enroll in multiple courses, and each course can have multiple students enrolled.

## 1.2. Draw an ER Diagram for This Hospital Scenario

A doctor can see multiple patients, but each patient can only be seen by one doctor.
Each patient has a unique patient ID, name, and date of birth.
Each doctor has a unique doctor ID, name, and specialty.

## 1.3. Draw an ER Diagram for This Scenario

Each book has a unique ID, title, and author.
Each member has a unique ID, name, and membership date.
A book can be borrowed by multiple members, and a member can borrow multiple books.
