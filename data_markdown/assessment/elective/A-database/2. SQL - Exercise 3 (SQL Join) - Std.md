# SQL Queries

## 1.6. Find the Names of Customers with Orders that have a Status of 'Shipped'

### Tables:
- **Orders** (OrderID, CustomerID, OrderDate, Status)
- **Customers** (CustomerID, CustomerName)

### SQL Query:
```sql
SELECT Customers.CustomerName
FROM Customers
JOIN Orders ON Customers.CustomerID = Orders.CustomerID
WHERE Orders.Status = 'SHIPPED';
```

## 1.7. Find the Names of Customers Who Placed Orders Between January 1, 2024, and March 31, 2024

### Tables:
- **Orders** (OrderID, CustomerID, OrderDate)
- **Customers** (CustomerID, CustomerName)

### SQL Query:
```sql
SELECT Customers.CustomerName
FROM Customers
JOIN Orders ON Customers.CustomerID = Orders.CustomerID
WHERE Orders.OrderDate BETWEEN '2024-01-01' AND '2024-03-31';
```

## 1.8. Retrieve Employee Names and Their Department Names, Sorted by Employee Name in Ascending Order

### Tables:
- **Employees** (EmpID, Name, DeptID)
- **Departments** (DeptID, DeptName)

### SQL Query:
```sql
SELECT Employees.Name, Departments.DeptName
FROM Employees
JOIN Departments ON Employees.DeptID = Departments.DeptID
ORDER BY Employees.Name ASC;
```

## 1.9. Find Product Names and Their Categories for Products in the 'Electronics' Category that Include 'Laptop' in the Product Name

### Tables:
- **Products** (ProductID, ProductName, CategoryID)
- **Categories** (CategoryID, CategoryName)

### SQL Query:
```sql
SELECT Products.ProductName, Categories.CategoryName
FROM Products
JOIN Categories ON Products.CategoryID = Categories.CategoryID
WHERE Categories.CategoryName = 'ELECTRONICS' AND Products.ProductName LIKE '%LAPTOP%';
```
