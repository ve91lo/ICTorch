## Queries

### 1.6. Retrieve the Names of All Products That Have Not Been Sold
```sql
SELECT ProductName
FROM Products
WHERE ProductID NOT IN (SELECT ProductID FROM Sales);
```

### 1.7. Retrieve the Names of Customers Who Have Made More Than 3 Orders but Have Never Ordered a Product from the 'Electronics' Category
```sql
SELECT CustomerName
FROM Customers
WHERE CustomerID IN (
SELECT CustomerID
FROM Orders
GROUP BY CustomerID
HAVING COUNT(OrderID) > 3
) AND CustomerID NOT IN (
SELECT DISTINCT o.CustomerID
FROM Orders o
JOIN Products p ON o.ProductID = p.ProductID
JOIN Categories c ON p.CategoryID = c.CategoryID
WHERE c.CategoryName = 'Electronics'
);
```
